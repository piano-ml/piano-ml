Why we built PianoML
Why we build PianoML

Learning Piano require a strong personal investment. It is a long and difficult process. The best piano learning app can't replace a real piano teacher. Using an app is not enough to learn piano and can lead you to bad habits.
That said practice is good. Practicing scale is one of the best way to learn Piano and this is why we built PianoML.
Each week a new scale to be practiced every day, according to a set of different exercises. Don't forget to always use a metronome when practicing.

What the software does
What the software does

PianoML will generate Major, Minor and other scales starting from any note.

Software will wait for you to play the note and give you instant feedback, you can select left, right or both hand you want to work.

By opening a MIDI file, you can choose one or two tracks to be transcribed into a Grand Staff score, with the option for hand separation.

You can select a range of measures and work in loop on them.

How does it works
How does it works

PianoML is a web application that run on your browser. There is no server. Thus we do not require you to get an account to run the app. All what you need is a web browser plugged to your Midi Keyboard.

By making the software open source, we want to give the opportunity to anyone to contribute to the project and make piano learning more accessible to a wider audience.
